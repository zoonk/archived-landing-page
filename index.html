Open-source solutions for education

Everything we do requires us to learn something but learning isn't always easy or fun. We want to change that.

We're building open-source tools to improve our experience while learning things.

Learning should be fun.
Learning should be practical.
Learning should be easy.
Learning should be accessible.

By improving how we learn things, we want to empower a new generation of makers because if we fix education, then we can fix everything else.
